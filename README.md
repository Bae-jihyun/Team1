### 실행 방법

1. 가상 환경 실행
    
    ```
    Windows: source myvenv/Scripts/activate
    
    Mac: source myvenv/bin/activate
    ```

2. 장고 설치

    ```
    pip install django
    ```
  
3. 서버 실행

    ```
    python manage.py runserver
    ```
